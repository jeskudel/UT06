Eventos del elemento HTML
¿Quiéres añadir algún evento al elemento?

En función de si este checkbox esta o no seleccionado, aqui debereis mostrar (los podeis poner ocultos o crearlos en el momento) elementos de un formulario que dejen añadir lo necesario para poder poner eventos asociados a estos elementos (podran ser de raton, de teclado, ...) y podran definir la acción que ocurrira cuando pase ese evento.

Esta parte la podeis implementar como querais, con textareas para las acciones a realizar para los eventos, con desplegables para el tipo de evento (click, mouseover, ...)
